# MNIST_Demo

这是一个给别人演示卷积神经网络如何运行的小demo, 代码基于Tensorflow。通过Tensorflow打造了两个小型的模型：回归模型和卷积神经网络。其中卷积神经网络可以达到94.84%的准确率。

#### 依赖的第三方库：

* Numpy
* tensorflow
* matplotlib
